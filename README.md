# myore

`myore` is an unofficial, community-made open-source alternative client for the ORE protocol.

## Homepage UI included in this repository

- Ore.supply-inspired desktop-first shell with sticky header and responsive behavior
- Sticky left shared-chat rail with mock messages and pinned composer bar
- Center board area with 4 stat cards (including highlighted Motherlode treatment)
- 5x5 mining tile grid with hover and selection states
- Right deploy panel with manual/auto segmented control, quick amount buttons, amount input, summaries, and deploy CTA
- Minimal centered footer links

## Local development

```bash
npm install
npm run dev
```

## Build and lint

```bash
npm run lint
npm run build
```
